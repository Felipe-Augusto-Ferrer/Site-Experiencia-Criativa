# 🧠 Experiência Criativa — Explorando Computação e IA

> Site interativo desenvolvido para a disciplina **Experiência Criativa: Explorando Computação e IA**, com o objetivo de aprender, na prática, como construir páginas web com **HTML, CSS e JavaScript** — utilizando IA como apoio durante o processo de criação e desenvolvimento.

---

## 📖 Sobre o projeto

Este projeto nasceu de uma pergunta simples: *e se o próprio código fosse tratado como material de criação, assim como tinta, papel ou som?*

A partir disso, o site foi pensado como um **ateliê digital**, onde cada seção é um experimento independente que explora uma forma diferente de interação entre computação, arte e som — tudo construído com tecnologias web puras, sem frameworks externos, para que o processo de criação ficasse o mais transparente e didático possível.

A inteligência artificial foi usada como **parceira no processo de aprendizagem**: ajudando a entender boas práticas de HTML semântico, organização de CSS, lógica de programação em JavaScript (Canvas API, Web Audio API) e princípios de design visual — sempre com o código sendo revisado, entendido e adaptado durante o desenvolvimento.

## ✨ O que você vai encontrar no site

| Seção | Descrição |
|---|---|
| 🌐 **Hero** | Rede de pontos generativa em `<canvas>` que reage ao movimento do mouse |
| 📝 **Sobre** | Apresentação do conceito e dos objetivos do projeto |
| 🎨 **Galeria generativa** | 4 algoritmos visuais diferentes (campo de fluxo, árvore recursiva, autômato celular, órbitas harmônicas) — clique para gerar uma nova versão |
| 🖼️ **Inspirações visuais** | Fotografias relacionadas aos temas de código, redes e inteligência artificial |
| 🎵 **Sequenciador sonoro** | Grade de 16 passos onde o som é **sintetizado em tempo real** pelo navegador, sem nenhum arquivo de áudio |
| 🎬 **Vídeo** | Espaço reservado para registro do processo, com animação de espera caso o vídeo ainda não tenha sido adicionado |
| ✏️ **Laboratório interativo** | Tela de desenho livre onde cada traço também dispara uma nota musical |

---

## 🛠️ Tecnologias utilizadas

- **HTML5** semântico
- **CSS3** (variáveis customizadas, Grid, animações, responsividade)
- **JavaScript puro** — Canvas API e Web Audio API, sem bibliotecas externas
- **Google Fonts**: Space Grotesk, IBM Plex Sans, JetBrains Mono
- **Lorem Picsum**: fotografias reais sob licença livre, usadas na galeria e nas inspirações visuais

## 📁 Estrutura de arquivos

```
site/
├── index.html      → estrutura e conteúdo das seções
├── style.css       → identidade visual (cores, tipografia, layout)
├── script.js       → toda a interatividade (canvas, som, desenho)
├── assets/         → pasta para imagens, vídeo e áudio próprios
└── README.md       → este arquivo
```

## 🚀 Como rodar localmente

1. Baixe ou clone este repositório
2. Abra a pasta no **VS Code**
3. Instale a extensão **Live Server**
4. Clique com o botão direito em `index.html` → **Open with Live Server**

O site abre no navegador e atualiza automaticamente a cada alteração salva.

## 🎯 Personalizações possíveis

- **Imagens próprias**: substitua o `src` de qualquer `<img>` por `assets/sua-imagem.jpg`
- **Vídeo próprio**: adicione o arquivo em `assets/video-processo.mp4`
- **Áudio gravado**: adicione um elemento `<audio>` na seção de som
- **Cores e fontes**: edite as variáveis no início do `style.css` (`:root { ... }`)
