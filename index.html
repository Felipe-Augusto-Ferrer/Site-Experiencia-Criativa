01

Sobre o projeto

Este site reúne os experimentos da disciplina Experiência Criativa: Explorando Computação e IA. A proposta é simples: tratar o código como material de criação — assim como tinta, papel ou som — e usar a tela do navegador como ateliê. Cada seção abaixo é uma peça independente, construída em HTML, CSS e JavaScript puro, sem bibliotecas externas pesadas, para que o processo fique visível e editável.

  • 01 Visuais generativos desenhados em tempo real no canvas
  • 02 Som sintetizado por código com a Web Audio API
  • 03 Vídeo como camada de contexto e registro do processo
  • 04 Um laboratório interativo para o público desenhar
02

Galeria generativa

cada quadro é desenhado por um algoritmo diferente. clique em um quadro para gerar uma nova versão.
